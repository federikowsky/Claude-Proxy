    **In `llm_proxy/infrastructure/config.py` — `ProviderSettings`:**
    Add after `finish_reason_map`:
    ```python
    retry_attempts: int = Field(default=2, ge=0, le=10)
    retry_backoff_base: float = Field(default=1.0, gt=0)
    retry_on_status: tuple[int, ...] = (429, 502, 503, 529)
    ```
    - `retry_attempts`: Number of retry attempts on transient errors (0 = no retries).
    - `retry_backoff_base`: Base delay in seconds for exponential backoff (delay = base * 2^attempt).
    - `retry_on_status`: HTTP status codes that trigger a retry.

    **In `llm_proxy/infrastructure/config.py` — `ModelSettings`:**
    Add after `generic_tool_emulation_policy`:
    ```python
    aliases: tuple[str, ...] = ()
    ```
    - `aliases`: Alternative names for this model. When a client requests any alias,
      it resolves to this model. Example: `aliases: ["claude-sonnet-4", "sonnet"]`
      allows clients to request either name.

    **In `llm_proxy/infrastructure/config.py` — `Settings.validate_cross_references`:**
    Add alias uniqueness validation after the existing model provider checks:
    ```python
    seen_aliases: dict[str, str] = {}
    for model_name, model in self.models.items():
        for alias in model.aliases:
            if alias in self.models:
                raise ValueError(f"alias '{alias}' on model '{model_name}' conflicts with a model name")
            if alias in seen_aliases:
                raise ValueError(
                    f"alias '{alias}' on model '{model_name}' conflicts with alias on '{seen_aliases[alias]}'"
                )
            seen_aliases[alias] = model_name
    ```

    **In `llm_proxy/infrastructure/resolvers.py` — `StaticModelResolver.__init__`:**
    Build an alias lookup dict at init time:
    ```python
    def __init__(self, settings: Settings) -> None:
        self._settings = settings
        self._alias_map: dict[str, str] = {}
        for model_name, model_cfg in settings.models.items():
            for alias in model_cfg.aliases:
                self._alias_map[alias] = model_name
    ```

    **In `StaticModelResolver.resolve()`:**
    After `model_name = requested_model or ...`, before `config = self._settings.models.get(model_name)`:
    ```python
    # Resolve alias to canonical model name
    model_name = self._alias_map.get(model_name, model_name)
    ```
    This way, if the requested model matches an alias, it's transparently resolved.

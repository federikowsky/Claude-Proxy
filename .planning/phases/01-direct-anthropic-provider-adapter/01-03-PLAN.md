**Fixture helpers (module-level):**

`_request() -> ChatRequest` — same as openrouter test but with `model="claude-sonnet-4-20250514"` (not `anthropic/claude-sonnet-4`).

`_model(**overrides) -> ModelInfo` — same pattern but with `provider="anthropic"` and `name="claude-sonnet-4-20250514"`.

`async def _chunks(data: bytes) -> AsyncIterator[bytes]:` — yields `chunk_bytes(data, 11)`.

**Required tests (each is a separate test function):**

1. **`test_translator_maps_full_request_payload`** — Create translator, call `to_payload(_request(), _model())`. Assert: `payload["model"] == "claude-sonnet-4-20250514"`, `payload["messages"]` is list with correct structure, `payload["max_tokens"] == 64`, `payload["stream"] is True`, `payload["tools"][0]["name"] == "bash"`, presence of `metadata`, `temperature`, `top_p`, `stop_sequences`. Assert `extensions` keys are forwarded.

2. **`test_translator_maps_count_tokens_payload`** — Create translator, call `to_count_tokens_payload()` with a request that has `thinking=ThinkingConfig(type="enabled", budget_tokens=2048)`. Assert: payload has `model`, `messages`, `thinking`. Assert: payload does NOT have keys `stream`, `max_tokens`, `temperature`, `top_p`, `stop_sequences`, `metadata`. This verifies the native count_tokens endpoint gets a clean subset.

3. **`test_translator_count_tokens_payload_includes_tools`** — Verify `to_count_tokens_payload()` includes `tools` and `tool_choice` when present on the request.

4. **`test_stream_normalizer_basic_text_flow`** — Feed SSE messages through parser+normalizer for a basic text response: `message_start` → `content_block_start(type=text)` → `content_block_delta(type=text_delta)` → `content_block_stop` → `message_delta(stop_reason=end_turn)` → `message_stop`. Assert event types in order. Assert `message_start` has correct `ChatResponse` fields. Assert `content_block_delta` has `TextDelta`. Assert `message_delta` has `stop_reason == "end_turn"`.

5. **`test_stream_normalizer_tool_use_flow`** — SSE sequence with `content_block_start(type=tool_use)` → `content_block_delta(type=input_json_delta)` → `content_block_stop`. Assert `ContentBlockStartEvent.block` is `ToolUseBlock`. Assert delta is `InputJsonDelta`.

6. **`test_stream_normalizer_thinking_flow`** — SSE sequence with `content_block_start(type=thinking)` → `content_block_delta(type=thinking_delta)` → `content_block_delta(type=signature_delta)` → `content_block_stop`. Assert deltas are `ThinkingDelta` and `SignatureDelta`.

7. **`test_stream_normalizer_no_done_sentinel`** — Create normalizer, verify that `normalize(SseMessage(event=None, data="[DONE]"))` does NOT return `MessageStopEvent` — it should raise `ProviderProtocolError` (invalid JSON) or return a warning. This is the critical D-03 test: Anthropic normalizer must NOT treat `[DONE]` as a valid termination signal.

8. **`test_stream_normalizer_terminates_on_message_stop`** — Feed `message_start` then `message_stop`. Assert last event is `MessageStopEvent`. Feed another `message_stop` — assert returns `None` (idempotent via `_emit_message_stop_once`).

9. **`test_stream_normalizer_error_event`** — Feed `SseMessage(event="error", data='{"type":"error","error":{"type":"overloaded_error","message":"Overloaded"}}')`. Assert returns `ErrorEvent` with `message="Overloaded"` and `error_type="overloaded_error"`.

10. **`test_stream_normalizer_ping`** — Feed `SseMessage(event="ping", data='{"type":"ping"}')`. Assert returns `PingEvent`.

11. **`test_stream_normalizer_cache_usage`** — Feed `message_start` with `usage` containing `cache_creation_input_tokens` and `cache_read_input_tokens`. Assert `MessageStartEvent.message.usage.cache_creation_input_tokens` and `cache_read_input_tokens` are correctly parsed.

12. **`test_stream_normalizer_unknown_event`** — Feed `SseMessage(event="unknown_type", data='{"type":"unknown_type"}')`. Assert returns `ProviderWarningEvent`.

All tests: use `@pytest.mark.asyncio` for async tests. Match the exact assertion style from `test_openrouter_adapter.py` (isinstance checks, attribute comparisons).
